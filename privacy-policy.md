# Privacy Policy – Bíblia Sagrada Online

**Last updated:** 2026

---

## 1. Introduction

**Bíblia Sagrada Online** is a Chrome extension designed to allow users to read the Holy Bible online in a simple, fast, and intuitive way.

We respect your privacy and are committed to protecting it.

---

## 2. Data Collection

The extension **does not collect, store, or share any personal data**.

We do **not** collect:
- Name
- Email address
- Physical or precise location
- Browsing history
- Personal identifiers
- Authentication data

---

## 3. Local Storage Usage

The extension uses local browser storage (`chrome.storage` and/or IndexedDB) **only on the user’s device** to store:

- User preferences (theme, font size, selected language/version)
- Reading state (last opened book and chapter)
- Highlighted verses and local annotations
- Cached Bible content to improve performance and reduce repeated network requests

This data **never leaves the user’s browser** and is not shared with any external service.

---

## 4. External Services

The extension performs **read-only HTTP requests** to a Bible API endpoint in order to retrieve:

- Books
- Chapters
- Verses

Important notes:
- Only static text data is retrieved
- No executable code is downloaded or executed
- No personal information is transmitted to the API

---

## 5. Data Sharing

We do **not** sell, rent, share, or transfer any data to third parties.

---

## 6. Security

All application logic is packaged locally within the extension.

- No remote scripts are executed
- No dynamic code execution (`eval`, `Function`, etc.) is used
- No content scripts inject or modify external websites

---

## 7. Changes to This Policy

This Privacy Policy may be updated if the extension’s functionality changes.

Any updates will be reflected on this page with a revised “Last updated” date.

---

## 8. Contact

If you have any questions regarding this Privacy Policy, please contact the developer through the Chrome Web Store listing.
